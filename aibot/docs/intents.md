# Daftar Intent (MVP)

| Intent | Deskripsi |
|--------|-----------|
| `check_laundry_status` | Cari transaksi laundry berdasarkan ID |
| `check_ticket_status` | Cari tiket berdasarkan ID (read-only) |
| `get_outlet_services` | Tanya layanan apa saja yang tersedia di outlet |
| `get_operating_hours` | Ambil jam operasional outlet |
| `get_active_promos` | Lihat promo yang sedang berlangsung |
| `get_member_info` | Informasi umum member |
| `talk_to_agent` | Eskalasi ke agen manusia |
| `unknown` | Fallback saat intent tidak dikenali |

## Di Luar Scope (MVP)

- Info pembayaran
- Pembuatan/perubahan tiket
- Refund
